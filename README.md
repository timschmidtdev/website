# Tim Schmidt - Personal Website

Personal website built with React, TypeScript, and Vite.

Visit: [timschmidt.dev](https://timschmidt.dev)

## Features

- Clean, minimalist design
- Light/Dark mode with system preference detection
- Fully responsive (mobile & desktop)
- Contact form integration
- Fast and lightweight

## Development

```bash
npm install
npm run dev
```

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions when you push to the `main` branch.

The workflow builds the site and deploys the `dist/` folder to GitHub Pages. No manual deployment steps needed!
